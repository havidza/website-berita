A. PETUNJUK INSTALASI & KONFIGURASI
	1. Install XAMPP terlebih dahulu pada PC/LAPTOP
	2. Pindahkan semua folder PERTEMUAN 8 ke folder htdocs xampp
		Alamat = Windows/Drive C -> Xampp -> htdocs
	3. Start apache + mysql pada Xampp.
	4. Import database blog.sql di phpMyAdmin
	5. Buka browser , lalu ketikan localhost/pertemuan14.
	6. Lalu pilih file/folder yang akan dibuka.
	7. Jika ingin membuka admin panel ketikan di browser
	   localhost/(nama-folder)/admin

B. STRUKTUR PROGRAM
	Di program ini terdapat 8 folder di dalamnya yaitu
	1. Folder Admin ( Berisi file admin panel )
		a. Berita
		b. Berita-jenis
		c. Penulis
		d. login
	2. Folder Config ( Berisi file koneksi ke database )
	3. Folder CSS 
	4. Folder Images
	5. Folder JS ( Berisi javascript & jQuery )
	6. Folder log 
	7. Folder Pages  ( Berisi halaman tampilan pengguna )
	8. Folder Vendor ( Berisi framework yang dipakai )
		a. Bootstrap
		b. tinymce
